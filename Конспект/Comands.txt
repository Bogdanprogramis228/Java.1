toLowerCase() - можна перетворити всі символи рядка на малі (рядкові);

toUpperCase() - можна перетворити всі символи рядка на великі (заголовні);

String str = console.nextLine(); - Коли програма дійде до виконання цього рядка, вона зупиниться й очікуватиме, поки користувач уведе дані й натисне клавішу Enter. Після цього все, що ввів користувач, буде збережено в змінній str;

int number = console.nextInt(); - Щоб зчитати з клавіатури число;
double number = console.nextDouble();

hasNextInt() - Там тип int? Чи можна його перетворити на тип int?
hasNextLine() - Там є ще один рядок?
nextInt() - Зчитує дані та перетворює їх на тип int
nextLine() - Зчитує весь рядок
nextFloat() - Зчитує дані та перетворює їх на тип float
